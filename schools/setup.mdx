## Firewall configuration

In order for your students to access the SEP Therapy Platform, the following ports will need to be configured.

| Protocol | Ports | Destination                         | Explanation          |
| -------- | ----- | ----------------------------------- | -------------------- |
| TCP      | 443   | \*.septherapy.com                   | general access       |
| TCP      | 443   | \*.firebaseio.com \*.googleapis.com | teletherapy services |
