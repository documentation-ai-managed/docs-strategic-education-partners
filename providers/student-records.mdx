---
title: Student records
---
Student records are used in the therapy room to link incoming participants to student records so that session time can be tracked over time. While you can create new student records in your therapy room when a student requests to be admitted, we recommend adding your caseload ahead of time.

## Accessing student records

To access your student records, click on the **Student records** link in the main navigation. On this page, you will see all of the student records you have created for the platform. Student records are not shared between providers and only you will be able to see the student records that you have created.&#x20;

## Adding new student records

To add a new student record, click on the **Add new student record** button in the upper right hand corner. Fill out the student's name, date of birth, and select the school they attend.&#x20;

<Callout kind="info" collapsed="false">
  If you do not see a school you are working with, please contact <a href="mailto:support@septherapy.com" target="_blank" rel="noopener noreferrer">support\@septherapy.com</a>.
</Callout>

Click the **Create new student** when you are done. The newly created record will show up in the table of all student records you have created.

### Adding new student records in the therapy room

You can also add a new student record in the therapy room when admitting a new participant (see <a href="/providers/admitting-students" target="_blank" rel="noopener noreferrer">Admitting students</a> for more details). This is helpful in case you are meeting with a new student and did not have time to create a student record ahead of time.

## Editing student records

To edit a student record, click on the three dots menu icon in the actions column of the student records table and select **Edit**. You can also click on the **Edit student record** button while viewing any individual student record.

## Viewing student records

To view a student record, click on the three dots menu icon in the actions column of the student records table and select **View details**. This page contains the basic details for the student record, the status of the student record, and the most recent sessions you have had with this student.

### Active vs. inactive student records

While you will be able to see all of your student records on the Student records page, only student records set to active will appear in the list of student records you can link to incoming participants in the therapy room (see <a href="/providers/admitting-students" target="_blank" rel="noopener noreferrer">Admitting students</a> for more details).

<Callout kind="tip" collapsed="false">
  If you know you will no longer be seeing a student, we recommend you set their student record to inactive. You will still have access to their session history and can set the student back to active to have it appear in the therapy room again.&#x20;
</Callout>

### Session history

Below the list of basic details about the student is a list of all the sessions you have had with the student along with the date, the start time, the end time, and the duration of the session.

The start time for each session is set once the **Confirm** button has been clicked when associating a student record with the incoming participant. The end time is set when you dismiss the student or they leave the session.&#x20;
